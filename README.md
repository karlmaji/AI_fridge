# AI_fridge
# 智能冰箱 调用百度语音识别API  
# 语音输入格式  
1.**东西** + **数量(必须包含数字）**  +存放 + **数字**+ **天/月/周**  
eg： 西瓜一公斤存放四天  
2. 删除 第 **数字** 行  或 删除所有行  
eg： 删除第一行
